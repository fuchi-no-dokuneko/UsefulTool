# Credits and Third-Party Software

UsefulTool's application pages are maintained in this repository. Three browser libraries are vendored for offline use:

- [Fabric.js](https://github.com/fabricjs/fabric.js) 7.4.0 - interactive canvas object model used by the layered image editor; MIT License. Local license: `vendor/LICENSE-fabric.txt`.
- [jsdiff](https://github.com/kpdecker/jsdiff) 9.0.0 - line comparison and unified-patch generation; BSD-3-Clause. Local license: `vendor/LICENSE-jsdiff.txt`.
- [pdf-lib](https://github.com/Hopding/pdf-lib) 1.17.1 - PDF loading, page copying, image embedding, and document creation; MIT License. Local license: `vendor/LICENSE-pdf-lib.md`.

The remaining tools use standard browser APIs such as Canvas, WebRTC, File, Blob, and Web Crypto without runtime CDN dependencies.
